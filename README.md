# 国际商务课程 (International Business)

## 课程简介

本课程基于查尔斯·希尔、托马斯·霍特（著），郭羽诞等译，《国际商务（第13版）》，中国人民大学出版社，2024。

课程介绍经济全球化背景下的企业管理变革，涵盖国际商务国别制度环境、贸易与投资环境、金融环境以及企业国际化战略与实践等专题。

## 课程文档

- [课程大纲](课程大纲.md) - 包含课程描述、先修课程、课程目的与任务的详细信息

## 课程目标

通过本课程的学习，学生将能够：

1. 树立全球经营视野和多元文化背景，了解企业国际化面临的环境、选择的方式和战略
2. 了解中国企业国际化的选择和发展路径
3. 理解管理基本原理和方法在国际经营环境下的变化和特殊性

详细的课程目标请参阅[课程大纲](课程大纲.md)。

## 培养目标

培养具有国际视野、通晓国际规则、知悉国际事务，能参与国际竞争的"有理想、能创新、敢承担"的企业管理高层次专业人才。
